Oppgave 3a:

Input taggen er en metode for en bruker å endre/sende inn data til en nettside og innenfor input taggen finnes det flere typer som text, date og submit.
Text typen lar bruker skrive inn all slags type text.
Date typen lar bruker skrive/legge inn en dato.
Submit typen lar en bruker submite/sende inn et form med andre inputs som text.

Oppgave 3b:

Det er en liten demo på starten av oppgave 3.

HTML:

<form action="Javascript:spraak()">
 <input type="text" id="input"><br><br>
 <input type="submit" id="submit">
</form>

JS:

function spraak(){
 let lang = document.getElementById("input").value;
 if (lang == "n"){
  document.getElementById("output").innerHTML = "Du er Norsk";
 } else if (lang == "s"){
  document.getElementById("output").innerHTML = "Du er Svansk";
 } else if (lang == "d"){
  document.getElementById("output").innerHTML = "Du er Dansk";
 } else {
  document.getElementById("output").innerHTML = "Skriv inn n for Norge, s for Sverige eller d for Danmark";
 }
}
